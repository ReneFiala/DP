**statistika.py**  
Vypočte statistické testy pro metriky na test setu a uloží do formátovaného XLSX. Vyžaduje SciPy, statsmodels a openpyxl. Konfigurace uvnitř skriptu.

**test_results/ **  
Vypočtené testovací metriky připravené pro statistické testování.

**vysledky_evaluace/ **  
Vypočtené výsledky statistických testů pro p=0,01 a 0,05.
